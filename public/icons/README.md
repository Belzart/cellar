# Icon Generation

Place the following icon files here for full PWA support:

| File | Size | Usage |
|------|------|-------|
| `icon-192.png` | 192×192 | Android home screen |
| `icon-512.png` | 512×512 | Splash screen |
| `icon-maskable-512.png` | 512×512 | Maskable (Android adaptive) |
| `apple-touch-icon.png` | 180×180 | iOS Add to Home Screen |
| `icon-180.png` | 180×180 | iPhone |
| `icon-167.png` | 167×167 | iPad Pro |
| `icon-152.png` | 152×152 | iPad |

## Quick Generation

Create a 512×512 PNG with your logo, then use one of these tools:
- https://maskable.app — test and generate maskable icons
- https://realfavicongenerator.net — generates all sizes from one source

## Design Guidelines

- Background: #0A0A0A (matches app theme)
- Use a wine glass or drop icon in #8B1A1A (burgundy)
- Maskable icons need 20% safe zone padding
